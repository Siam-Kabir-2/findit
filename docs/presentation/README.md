# FindIt Presentation

## PowerPoint

**Use this file:** [`FindIt_Presentation_v2.pptx`](FindIt_Presentation_v2.pptx)

Clean university deck (~41 slides): overview, workflow, architecture, UI screenshots, ER/schema, PL/SQL, setup, testing, conclusion.

White background, dark-blue accent, short captions.

### Rebuild

Close PowerPoint first, then:

```bash
py -3 make_visuals.py
py -3 build_pptx.py
```

Screenshots live in `screenshots/`.

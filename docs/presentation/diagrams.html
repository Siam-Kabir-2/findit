System Workflow — FindIt

flowchart TD
  A[Visitor opens website] --> B[Browse / search items]
  B --> C[Register or log in]
  C --> D[Report lost or found item]
  D --> E[(Oracle stores item)]
  E --> F[Another user submits claim]
  F --> G[Claim status = Pending]
  G --> H[Admin reviews claim]
  H --> I{Approve or Reject?}
  I -->|Approve| J[Claim Approved · Item Claimed]
  I -->|Reject| K[Claim Rejected]
  J --> L[User checks final result]
  K --> L
